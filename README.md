# Portfolio Website

Deployment-ready portfolio starter built with React, Vite, and Tailwind CSS.

## Stack

- React
- Vite
- Tailwind CSS
- Vercel-compatible static deployment

## Project Structure

```text
portfolio/
├── public/
├── src/
│   ├── components/
│   │   ├── layout/
│   │   ├── sections/
│   │   └── ui/
│   ├── data/
│   ├── App.jsx
│   ├── index.css
│   └── main.jsx
├── index.html
├── package.json
├── postcss.config.js
├── tailwind.config.js
├── vercel.json
└── vite.config.js
```

## Run Locally

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

1. Push the project to GitHub.
2. Import the repository into Vercel.
3. Keep the detected framework as `Vite`.
4. Deploy.
